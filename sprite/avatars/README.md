# avatars/ — プレイヤー用プリセットアイコン

PROFILE パネルに表示されるアバター画像を置くフォルダ。

## 追加のしかた

1. 画像をこのフォルダに入れる
2. `manifest.json` の `items` に 1 行追加

    ```json
    { "id": "skull", "file": "skull.png", "label": "SKULL" }
    ```

3. ゲームをリロード → PROFILE パネルの選択肢に出る

## 画像の推奨スペック

| 項目      | 推奨                                    |
|-----------|-----------------------------------------|
| フォーマット | **PNG** (透過対応)                    |
| サイズ    | **256×256** 正方形 (最低 128×128)       |
| 背景      | 透過 (アルファ) または単色              |
| ファイル名 | 英小文字 + アンダースコアのみ `xxx.png` |

## 注意

- ファイル自体を削除すると、そのアイコンを選んでいたセーブは自動で "NONE" に戻る (無効参照の防止)
- `id` は localStorage に保存されるので、一度公開したあとは変えない方が無難 (ユーザーの選択がリセットされる)
- 世界観 (VHS モノクロ + シアン / 赤アクセント) に寄せると全体の見栄えが揃う
